About

This project was inspired by me and my friends personal experience working in the pharmacy. I realized the need for an efficient system to manage Employe, inventory, sales, and other pharmacy-related operations. The Pharmacy Information Management System aims to create and deploye an automated pharmacy information management system which improves overall working load that happens on paper based pharmacy information management system.

This is a Portfolio Project for Holberton School, I developed this system as part of my coursework. You can find the project repository on GitHub:
